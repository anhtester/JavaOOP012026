# Java OOP Examples

Đây là kho lưu trữ chứa mã nguồn ví dụ cho việc học lập trình hướng đối tượng (OOP) trong Java, dành cho khóa học Selenium Java 01/2026.

## Cấu trúc dự án

- `DemoJavaOOP/`: Thư mục dự án chính
  - `src/Main.java`: Điểm vào của ứng dụng Java

## Cách bắt đầu

1. Sao chép kho lưu trữ về máy:
   ```
   git clone <repository-url>
   ```

2. Mở dự án trong IDE yêu thích (ví dụ: IntelliJ IDEA, Eclipse).

3. Chạy file `Main.java` để xem ví dụ.

## Yêu cầu hệ thống

- Java JDK 8 trở lên
- IDE hỗ trợ Java (khuyến nghị IntelliJ IDEA)
